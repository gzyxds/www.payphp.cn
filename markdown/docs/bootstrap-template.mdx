## 下载模板：

您可以通过访问您的账户，下载所有可用的模板，这些模板都包含在大包中。

只要您有我们的账户，您可以随时下载。

## **安装模板**：

要能够使用模板，首先，您需要安装它。以下是您需要采取的步骤来安装它：

**步骤#1:** 从 GrayGrids 下载模板。当您下载模板时，您将得到一个 zip 文件。

**步骤#2**: 解压/提取 zip 文件。解压后，您将找到所有您需要的文件和资产。

**步骤#3**: 现在，在代码编辑器中打开文件夹，并根据您的需求更新内容。

**步骤#4**: 更新内容后，现在转到您的 cPanel 或打开您的 FTP 客户端（如 Filezilla），并将模板上传到您的服务器。确保上传到根目录。

安装完成，现在转到 **yourdomainname.com/index.html**。您应该看到网站在线。

## **目录结构**

当您提取模板文件夹时，您将在文件夹内看到类似这样的内容：

```
|-- 模板文件夹
    |-- 资产
    |    |-- css
    |       |-- 所有 CSS 文件
    |    |-- scss
    |       |-- 所有 SCSS 文件
    |    |-- js
    |       |-- 所有 JS 文件
    |    |-- 字体
    |       |-- 所有字体文件
    |    |-- 图像
    |       |-- 所有图像
    |-- index.html 和所有 HTML 文件
```

## **定制**

### **更改默认主题颜色**

如果您想更改默认的主题颜色，打开 `资产->css>main.css` 文件。然后，找到默认颜色并用您偏好的颜色替换。

**注意：** 如果您熟悉 Scss，您可以通过更改 Scss 变量来更改颜色。

### **更改默认字体**

如果您想更改默认的字体，同样，转到 `资产->css>main.css`

然后，找到所有的字体系列，然后用您自己的字体系列替换其属性。

确保在您的 HTML 或 CSS 文件中包含字体的 URL。否则，它不会工作。

**注意：** 如果您熟悉 Scss，您可以通过更改 Scss 变量来更改字体系列。
